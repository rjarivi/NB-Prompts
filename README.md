## üöÄ Getting Started

### Prerequisites

You need a modern web browser. That's it!

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/rjarivi/nano-banana-prompts.git
   ```

2. Navigate to the project directory:
   ```bash
   cd nano-banana-prompts
   ```

3. Open `index.html` in your browser.

### Running Locally (Optional)

If you prefer running a local server (recommended for better asset loading):

```bash
# Using Python
python3 -m http.server

# Using Node.js (npx)
npx serve .
```
